# Gris-Cat

Luxury Women's Fashion E-commerce Platform

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Fonts:** Playfair Display (Headings), Inter/Montserrat (Body)

## Brand Colors

- **Background:** #FFFFFF (White)
- **Primary Text:** #1A1A1A (Near Black)
- **Brand Accent:** #708090 (Slate Grey / Ash Grey)
- **Soft Accent:** #F5F5F0 (Warm Light Grey/Beige)

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
